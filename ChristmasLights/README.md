Christmas Lights
============================

This (seasonal) project was to address my desire to automate switching my Christmas lights on/off. It's pretty straight forward :)


Requirements:
-------------

1. Photoresistor
2. 10k resistor
3. PowerSwitch Tail II
4. DS1307 (Real Time Clock)


Photoresistor Circuit Diagram:
------------

        ----------------------------------------------------
                   PhotoR     10K
         +5    o---/\/\/-----/\/\/---o GND
        
        ----------------------------------------------------
